What is Kali Linux ?


Kali Linux is a GNU/Linux distribution based on Debian.It took the place of the backtrack os arround 2013.
The purpose of Kali Linux is to provide a distribution that can group all the tools needed for security testing, and pentesting of an operating system, including intrusion tests.



While you can install Kali Linux by replacing the existing operating system, using it via a virtual machine would be a better and safer option.




How To Get It ?


To get Kali Linux you have 2 solutions (most commons technics):


1.Install Kali Linux on pc :

-download kali linux's image file

-create a bootable usb and plug it into a laptop or a pc (when it's off).

-switch on the computer and go into the bios.

-then boot on the usb drive (from the boot menu in the bios)to install it ...

-that's it , you have kali linux on your computer ....

-PS : if you want to run kali linux without installing it select "kali linux live" when you booted on the usb key .



2.Install Kali Linux in a virtual machine :

-download kali linux's image file

-install a software like virtual-box, or vmware .

-then add a new VM with the kali linux img file

-you have your kali linux VM

-PS : put the mouse pointer on the image to make it appear.






Why Kali Linux Is The Most Used OS By The Pentesters?


Kali linux is the most used OS by the pentesters because of 3 things:


1.Kali Linux include a lot of tools which can be used for penstesting:

-Cracking, Wireless attacks, Exploitation Tools, Forensics Tools, Information Gathering ...

-The most used tool by pentesters remain Metasploit, wich include a lot of exploit for all types of Vulnerability


2.Kali Linux is a pretty stable distribution:

-for having tested other distributions like Parrot OS or Backtrack OS (The ancestor of Kali), the best alternative remain kali Linux (less bugs, a lot of programms included)


3.Kali Linux is the penstesting distribution with the best documentation:

-Far behind the other hacking distributions, Kali Linux is the most famous one; that's why it has also the best documentation. This documentation you allow to progress faster than with the other distributions and to upgrade it with some community developped tools (on github for exemple).


-PS : Kali Linux is a pretty simple distribution, but some linux bases are required to fully enjoy its tools.
